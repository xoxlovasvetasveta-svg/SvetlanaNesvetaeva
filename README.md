Разработала ПМИ и руководство пользователя для сайта бронирования капсульных отелей "Капсула"

Задача была разработать программу и методику испытаний, а также руководство пользователя для сайта бронирования капсульных отелей "Капсула"

Что сделано:
-	провела тестирование функциональности сайта «Капсулы» по плану тестирования: дополнила план тестирования недостающими тест-кейсами, провела ручное тестирование сайта по плану, сравнила ожидаемые результаты с фактическими https://disk.yandex.ru/i/3wzNrBxLhZ3MJg;
-	составила программу и методику испытаний сайта на основании проведенной работы по тестированию сайта https://disk.yandex.ru/i/9Yp2TGDTbGKQtg;
-	подготовила руководство пользователя в соответствии с ГОСТ 19, описывающее все функции, которые может выполнить пользователь на сайте https://disk.yandex.ru/i/3coEnGW2iRW1VQ.
